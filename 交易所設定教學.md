# 玩家交易所設定教學

1. 到 https://supabase.com 建立免費帳號與專案。
2. 進入 Authentication → Providers → Anonymous Sign-Ins，開啟匿名登入。
3. 進入 SQL Editor → New query，貼上 `supabase-market.sql` 全部內容並按 Run。
4. 進入 Project Settings → API，複製 Project URL 與 anon/public key。
5. 開啟 `supabase-config.js`，把 `YOUR_SUPABASE_URL` 與 `YOUR_SUPABASE_ANON_KEY` 換成剛才複製的值。
6. 用 GitHub Desktop Commit 並 Push，等待 GitHub Pages 更新。
7. 進入遊戲按右側「交易所」，第一次會自動建立匿名玩家帳號。

## 使用方式

- 錢包：先將遊戲金幣存入交易錢包。
- 掛賣：選擇背包內未鎖定裝備，設定數量與單價。
- 市場：瀏覽其他玩家商品並購買。
- 待領取：購買或下架後，必須在此領回物品。
- 賣出所得直接進入交易錢包，扣除 5% 手續費後可提領回遊戲。

## 目前安全限制

交易所的同時購買、超賣與錢包扣款由資料庫鎖定；但角色背包和遊戲金幣仍在瀏覽器 localStorage，熟悉開發工具的人仍可能修改或還原本機存檔。正式公開營運前，應再把角色背包、金幣與存檔搬到伺服器端。
